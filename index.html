📐 Formula Reference

Standard Form

y = ax² + bx + c

Line of Symmetry

x = -b/(2a)

Turning Point

x = -b/(2a)

y = f(-b/(2a))

Quadratic Formula

x = (-b ± √(b²-4ac))/(2a)

Discriminant

D = b² - 4ac

  • D > 0: Two real roots
  • D = 0: One repeated root
  • D < 0: No real roots

Vertex Form

y = a(x - h)² + k

Vertex: (h, k)
